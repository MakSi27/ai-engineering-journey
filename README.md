# 🚀 AI Engineering Journey

A structured, no-excuses 8-month journey to become a great AI Engineer — 
built from the ground up, one commit at a time.

## 🎯 Goal
Master AI engineering from mathematical foundations to production-ready 
systems — deeply, not superficially.

## 🗺️ Roadmap
- **Month 1** — Math (Linear Algebra, Calculus, Probability) + PyTorch Basics
- **Month 2** — Neural Networks + Computer Vision
- **Month 3** — Transformers + Attention Mechanism
- **Month 4** — LLMs + Fine-tuning + NLP
- **Month 5** — RAG Pipelines + Vector Databases
- **Month 6** — AI Agents + Deployment
- **Month 7** — Build & Ship 3 Portfolio Projects
